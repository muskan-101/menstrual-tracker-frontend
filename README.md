
# 🌸 Bloom in Every Cycle – Menstrual Tracker

A heartfelt and empathetic web-based **Menstrual Tracker** designed to help women track their menstrual cycles, predict their next period, and stay informed about menstrual health and PCOS/PCOD cautions.

---

## 💫 About the Project

**Bloom in Every Cycle** is more than just a tracker — it’s a space for women to understand their body better.  
It predicts upcoming periods, provides PCOS/PCOD guidance, and links to menstrual hygiene resources.  
The project was created using **HTML**, **CSS**, and touch of **JavaScript** to ensure a simple, elegant, and interactive experience.

---

## 💻 Tech Stack

- **Frontend:** HTML, CSS  
- **Scripting:** JavaScript (for navigation, interactivity and predicting next cycle)  
- **Deployment:** GitHub Pages / Vercel

---

## ✨ Features

- 🩸 Predict next period based on last period date and average cycle length  
- 🌷 PCOS/PCOD caution and guidance section  
- 💌 Registration and Login interface  
- 📖 Articles & Health Tips section  
- 💖 Contact form to reach out to the creator  
- 🌐 Smooth scrolling and responsive layout
